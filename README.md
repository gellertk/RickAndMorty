# RickAndMorty

Реализовать постраничную загрузку данных о персонажах по апи с кешированием картинок, при нажатии на строку списка должна появляться картинка персонажа и детальная информация о нем

**General**
* No Storyboard
* MVC

**Network**
* URLSession
